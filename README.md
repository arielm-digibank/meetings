# Project Name

Meetings

## Usage

start the Application: 

from ide:
run
        ```
        MeetingserviceApplication
        ```

from command line:
```
    mvn clean package
    java -jar target/meetingservice-0.0.1-SNAPSHOT.jar
```

open swagger: 
http://127.0.0.1:8080/swagger-ui.html
